# 🎬 Movie Data Analysis (Netflix Dataset)

## 📌 Project Overview

This project focuses on analyzing a Netflix movie dataset to uncover insights about content trends, genres, ratings, and production patterns. The goal is to understand how movies and shows are distributed across different categories and identify key patterns in the entertainment industry.


## 🎯 Objective

* Analyze Netflix movie and TV show data
* Identify trends in genres, release years, and ratings
* Explore country-wise and category-wise distribution
* Gain insights into content growth over time


## 🛠️ Tools & Technologies Used

* **Python 🐍**
* **Jupyter Notebook**
* **Pandas** – Data manipulation & cleaning
* **NumPy** – Numerical operations
* **Matplotlib** – Data visualization
* **Seaborn** – Advanced visualizations


## 📂 Dataset

* Netflix Movies & TV Shows Dataset
* Contains information like:

  * Title
  * Genre
  * Release Year
  * Country
  * Rating
  * Duration
  * Type (Movie/TV Show)


## 🔍 Key Analysis Performed

* 📊 Distribution of Movies vs TV Shows
* 🌍 Country-wise content analysis
* 🎭 Genre popularity analysis
* 📅 Year-wise content trend
* ⭐ Rating classification insights
* ⏱️ Duration analysis of movies


## 📈 Key Insights

* Movies dominate the Netflix catalog compared to TV shows
* Certain genres like Drama and Comedy are most popular
* Content production has increased significantly after 2015
* The United States contributes the largest share of content


## 📊 Sample Visualizations

* Bar charts for genre distribution
* Line graphs for yearly trends
* Count plots for ratings
* Heatmaps for correlations


## 🚀 How to Run This Project

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/movie-data-analysis.git
   ```

2. Navigate to the project folder:

   ```bash
   cd movie-data-analysis
   ```

3. Open Jupyter Notebook:

   ```bash
   jupyter notebook
   ```

4. Run the notebook file:

   ```
   Movie Data Analysis Netflix.ipynb
   ```
